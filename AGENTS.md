# OpenScience Agent Notes

- This repository is an OpenScience desktop fork based on upstream
  `anomalyco/opencode` tag `v1.17.11`.
- Keep the OpenCode provider/model flow native. Do not add Huanjing/Sim Studio
  login, balance, recharge, model proxy, API base injection, or shared API keys.
- OpenScience desktop identity lives in `packages/desktop`:
  product name `OpenScience`, appId/AUMID `ai.svd.openscience.desktop`, and
  protocol `openscience`.
- The bundled science layer lives under
  `packages/desktop/resources/openscience-config`.
- `skills-src/` contains OpenScience-owned skill sources.
- `skills/` is generated by
  `bun run --cwd packages/desktop materialize:skills` and should not be edited
  by hand.
- External skill source revisions are recorded in
  `packages/desktop/resources/openscience-config/skills.lock.json`.
- The first release target is Windows x64 NSIS via GitHub Releases. Builds are
  unsigned unless a future release explicitly adds signing.
- Prefer `rg` for search. Use `apply_patch` for manual edits.
